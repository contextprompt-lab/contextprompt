## Dependency Audit (Monday)

**3 high-severity vulnerabilities found.**

| Package | Version | Severity | Via | Production? | Fix Available |
|---------|---------|----------|-----|-------------|---------------|
| `path-to-regexp` | 8.3.0 | **High** | `express@5.2.1 → router@2.2.0` | **YES** | Yes |
| `vite` | 7.3.1 | High | `vitest@3.2.4` | No (dev only) | Yes |
| `picomatch` | 4.0.3 | High | `tsup@8.5.1`, `vitest@3.2.4` | No (dev only) | Yes |

### Risk Assessment

**`path-to-regexp@8.3.0` (PRODUCTION — most critical)**
- CVEs: GHSA-j3q9-mxjg-w52f (ReDoS via sequential optional groups, CVSS 7.5) and GHSA-27v5-c462-wpq7 (ReDoS via multiple wildcards)
- Impact: Express route matching could hang on crafted URLs — real DoS risk against the production server
- Mitigation: Fix is `path-to-regexp@8.4.0` (Express 5 router constraint)
- Status: **⚠️ Needs human action** — `npm audit fix` would add 49 packages (blocked by agent guardrails)

**`vite@7.3.1` (DEV ONLY — low actual risk)**
- CVEs: Path traversal in `.map` handling, `server.fs.deny` bypass, arbitrary file read via WebSocket
- Impact: Only applies when running Vite dev server — `vitest` uses vite internally but does NOT run a web server during `npm test`
- Production risk: **None** — vite is not in the production bundle

**`picomatch@4.0.3` (DEV ONLY — low actual risk)**
- CVEs: Method injection in POSIX character classes, ReDoS via extglob quantifiers
- Impact: Only affects build tools (`tsup`, `vitest`) — not reachable from production
- Production risk: **None**

### Action Required
`npm audit fix --dry-run` shows: **add 49 packages, change 3** — exceeds safe automated action threshold. The path-to-regexp production fix requires a human to either:
1. Run `npm audit fix` and verify tests still pass, or
2. Manually pin `express@latest` (which may ship with a fixed router), or
3. Add `overrides.path-to-regexp = "^8.4.0"` to `package.json`

**Option 3 (package.json overrides) is the safest minimal fix** — would update only the pinned version without adding 49 packages. Recommend human review.
